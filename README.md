# Tata-Data-Visualization-Forage
## Project Overview

This project was completed as part of the Tata Data Visualization Virtual Experience Program on Forage.

## Tools Used

* Power BI
* Excel Dataset

## Tasks Completed

### Task 1

Created a revenue trend visualization to analyze sales performance over time.

### Task 2

Compared revenue and quantity across countries while excluding the United Kingdom to identify international sales trends.

### Task 3

Identified top customers based on revenue contribution using Power BI filters and visualizations.

### Task 4

Presented key findings and business insights based on the created visualizations.

## Key Insights

* Revenue trends were analyzed over time.
* International market performance was compared.
* Top revenue-generating customers were identified.
* Business insights were communicated effectively through data visualization.
